# Product Overview

## Описание
Telegram Mini App для сопровождения общих проектов. Платформа поддерживает несколько независимых рабочих пространств (workspace), что позволяет использовать её для разных бизнесов или направлений в рамках одной системы.

## Рабочее пространство (Workspace)
Workspace — это изолированная площадка для одного бизнеса или направления.

Каждый workspace:
- имеет собственных администраторов
- имеет собственных помощников и клиентов
- имеет собственные проекты, мероприятия, программы и счета
- полностью изолирован от других workspace-ов

Примеры использования:
- несколько разных бизнесов одного владельца
- разные направления внутри одной компании
- несколько франшиз под одним управлением

## Базовая сущность
Базовая сущность системы: общий проект внутри workspace.

Проект может включать:
- одного или нескольких клиентов
- одного или нескольких помощников
- общую программу сопровождения
- отдельные направления работы по ролям или специализациям

Пример:
- workspace "Семейное агентство Иванова"
- проект "Семейное сопровождение Петровых"
- клиенты: муж и жена
- помощники: координатор, помощник по медицинским вопросам, помощник по документам

Внутри одного проекта часть данных общая для всех участников, а часть привязана к конкретному клиенту или направлению работы.

## Роли

### Системный уровень
- super_admin — управляет всеми workspace-ами, создаёт новые площадки, назначает workspace_admin-ов

### Уровень workspace
- workspace_admin — полный контроль внутри своей площадки
- assistant — работает с проектами в рамках своего workspace
- client — видит свои проекты и оплаты

Один пользователь Telegram может иметь разные роли в разных workspace-ах.

## Важно
Система НЕ является медицинской.

Не хранит:
- диагнозы
- назначения лечения
- медицинские документы

Хранит только:
- состав проекта
- роли участников проекта
- этапы
- статусы
- оплаты

## Основная ценность
- единая рабочая зона по проекту
- прозрачность ролей и зон ответственности
- прозрачность оплат
- контроль прогресса
- удобство работы помощников
- возможность масштабировать на несколько бизнесов без отдельных систем
