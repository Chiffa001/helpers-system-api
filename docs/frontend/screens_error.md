# Экраны ошибок

Единый переиспользуемый экран, который появляется вместо основного контента при критических сбоях.
Отображается по центру экрана на нейтральном фоне.

---

## Структура экрана

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│          [иконка 48px]          │
│                                 │
│           Заголовок             │
│                                 │
│        Описание ошибки          │
│     в одну или две строки       │
│                                 │
│        [ Основное действие ]    │
│      [ Вторичное действие ]     │  ← если нужно
│                                 │
└─────────────────────────────────┘
```

- иконка: 48×48px, цвет `hint` (серый)
- заголовок: крупный текст, `text-primary`
- описание: мелкий текст, `text-hint`, до 2–3 строк
- кнопки: по центру, с отступом от описания 24px

---

## Сценарии

### 1. Нет соединения

**Когда появляется:** нет интернета или запрос не дошёл до сервера.

| | |
|---|---|
| Иконка | `wifi-off` |
| Заголовок | Нет соединения |
| Описание | Проверьте подключение к интернету и попробуйте ещё раз. |
| Кнопка | Повторить (primary) |

---

### 2. Ошибка сервера

**Когда появляется:** сервер вернул ошибку (500, 502, 503).

| | |
|---|---|
| Иконка | `alert-circle` |
| Заголовок | Что-то пошло не так |
| Описание | Мы уже в курсе проблемы. Попробуйте через несколько минут. |
| Кнопка | Повторить (primary) |

---

### 3. Сессия завершена

**Когда появляется:** токен авторизации истёк или стал недействительным.

| | |
|---|---|
| Иконка | `lock` |
| Заголовок | Сессия завершена |
| Описание | Для продолжения работы необходимо войти снова. |
| Кнопка | Войти (primary) |

---

### 4. Нет доступа

**Когда появляется:** у пользователя недостаточно прав для этого раздела.

| | |
|---|---|
| Иконка | `shield-off` |
| Заголовок | Нет доступа |
| Описание | У вас недостаточно прав для просмотра этой страницы. |
| Кнопка | На главную (primary) |

---

### 5. Не найдено

**Когда появляется:** ресурс был удалён или ссылка недействительна (workspace, проект, мероприятие).

| | |
|---|---|
| Иконка | `search-x` |
| Заголовок | Не найдено |
| Описание | Запрашиваемый раздел не существует или был удалён. |
| Кнопки | Назад (secondary) · На главную (primary) |

---

### 6. Пространство приостановлено

**Когда появляется:** workspace временно заблокирован администратором платформы.

| | |
|---|---|
| Иконка | `pause-circle` |
| Заголовок | Пространство приостановлено |
| Описание | Доступ временно ограничен администратором платформы. Для уточнения деталей обратитесь к вашему администратору. |
| Кнопка | Выбрать другое пространство (primary) |

> Администратор workspace дополнительно видит кнопку **Управление пространством** (secondary).

---

### 7. Пространство архивировано

**Когда появляется:** workspace больше не активен.

| | |
|---|---|
| Иконка | `archive` |
| Заголовок | Пространство архивировано |
| Описание | Это рабочее пространство больше не активно. |
| Кнопка | Выбрать другое пространство (primary) |

---

### 8. Запущено не в Telegram

**Когда появляется:** пользователь открыл ссылку в браузере, а не в Telegram.

| | |
|---|---|
| Иконка | логотип Telegram |
| Заголовок | Откройте в Telegram |
| Описание | Это приложение работает только внутри Telegram. |
| Кнопка | Открыть Telegram (primary) |

---

## Иконки

Все иконки — из библиотеки [Lucide](https://lucide.dev), кроме логотипа Telegram.

| Сценарий | Иконка Lucide |
|----------|--------------|
| Нет соединения | `WifiOff` |
| Ошибка сервера | `AlertCircle` |
| Сессия завершена | `Lock` |
| Нет доступа | `ShieldOff` |
| Не найдено | `SearchX` |
| Приостановлено | `PauseCircle` |
| Архивировано | `Archive` |
| Не в Telegram | логотип Telegram |
