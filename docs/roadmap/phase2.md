# Phase 2

## Workspace-уровень

- настройки workspace (название, slug, логотип)
- отдельный Telegram-бот на каждый workspace
- приглашение пользователей по ссылке в workspace

## Биллинг (Phase 2)

- оплата подписки через встроенный провайдер (Stripe / ЮKassa)
- автопродление подписки
- уведомления об истечении подписки
- дашборд комиссий для super_admin
- возможность workspace_admin видеть удержанные комиссии и сравнивать с тарифом подписки

---

## Улучшения

- React админка (super_admin и workspace_admin)
- аналитика по workspace
- частичные оплаты
- мультивалютность
- шаблоны программ (переиспользование между проектами)

---

## Расширения

- уведомления (push через Telegram Bot)
- отчеты по workspace
- экспорт данных

---

## Масштабирование

- гибкие права доступа внутри проекта
- несколько активных проектов на одного клиента
- роли менеджеров (промежуточная роль между admin и assistant)
- межworkspace-ные аналитика и отчеты для super_admin
